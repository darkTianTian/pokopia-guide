# Pokopia 料理攻略：全食譜、工具與效果

料理是 Pokémon Pokopia 中最實用的系統之一。食物可以完全恢復你的 PP 並強化特定招式，對探索和棲息地建設至關重要。本攻略涵蓋從解鎖料理到所有食譜及效果的完整內容。

## 如何解鎖料理

料理系統在**ゴツゴツやま（嶙峋山）**劇情中解鎖（第三座島嶼）。你會遇到**大廚 Dente**，一隻被困在桶裡的特殊貪心栗鼠。救出他後，他會教你如何烹飪強化能力的料理。從此你就可以在任何料理台使用烹飪功能。

## 烹飪工具

你需要在工作台上製作 4 種烹飪工具，每種工具對應不同類型的菜餚：

| 工具 | 材料 | 菜餚 |
|------|------|------|
| 砧板 | 木材 x2 | 沙拉 |
| 鍋子 | 銅棒 x2 | 湯品 |
| 平底鍋 | 銅棒 x2 | 漢堡排 |
| 麵包窯 | 火山灰 x4、石頭 x2、小圓木 x2 | 麵包 |

**重要：** 鍋子、平底鍋和麵包窯都需要點燃的火源才能使用。把它們放在篝火旁邊。

## 全食譜

### 沙拉（強化飛葉快刀）

沙拉是你學會的第一種料理。它們強化你的**飛葉快刀**招式，讓你能種植藤蔓、苔蘚和植物。

| 食譜 | 材料 | 口味 |
|------|------|------|
| 簡單沙拉 | 葉子 x1 + 其他材料 x1 | — |
| 海草沙拉 | 葉子 x1 + 海草 x1 | 苦味 |
| 麵包丁沙拉 | 葉子 x1 + 麵包 x1 | 辣味 |
| 橙橙果沙拉 | 葉子 x1 + 橙橙果 x1 | 甜味 |
| 切絲沙拉 | 葉子 x1 + 材料 x1 + 砍伐寶可夢 | 酸味 |
| 碎莓沙拉 | 葉子 x1 + 零餘果 x1 + 粉碎寶可夢 | 澀味 |

### 湯品（強化水槍）

湯品強化你的**水槍**招式，增加射程和水量。

| 食譜 | 材料 | 口味 |
|------|------|------|
| 簡單湯 | 清水 x1 + 其他材料 x2 | — |
| 海草湯 | 清水 x1 + 海草 x1 + 其他 x1 | 苦味 |
| 健康湯 | 清水 x1 + 豆子 x1 + 葉子 x1 | 辣味 |
| 蘑菇湯 | 清水 x1 + 蘑菇 x1 + 其他 x1 | 澀味 |

### 漢堡排（強化碎岩）

漢堡排強化你的**碎岩**招式，讓你能打碎更堅硬的岩石和材料。

| 食譜 | 材料 | 口味 |
|------|------|------|
| 簡單漢堡排 | 豆子 x1 + 其他材料 x3 | — |
| 番茄漢堡排 | 豆子 x1 + 番茄 x1 + 其他 x2 | 酸味 |
| 馬鈴薯漢堡排 | 豆子 x1 + 馬鈴薯 x1 + 其他 x2 | 甜味 |
| 蘑菇漢堡排 | 豆子 x1 + 蘑菇 x1 + 其他 x2 | 澀味 |

### 麵包（強化居合斬）

麵包強化你的**居合斬**招式，增加範圍和效果。

| 食譜 | 材料 | 口味 |
|------|------|------|
| 簡單麵包 | 小麥 x1 + 其他材料 x2 | — |
| 胡蘿蔔麵包 | 小麥 x1 + 胡蘿蔔 x1 + 其他 x1 | 辣味 |
| 橙橙果麵包 | 小麥 x1 + 橙橙果 x1 + 其他 x1 | 酸味 |
| 麵包碗 | 小麥 x1 + 湯 x1 + 其他 x1 + 燃燒寶可夢 | 辣味 |

## 料理效果

食用料理有兩大好處：

1. **完全恢復 PP** — 黃色 PP 量表完全回滿
2. **強化特定招式** — 每種菜餚強化不同的招式：
   - 沙拉 → 飛葉快刀（種植植物）
   - 湯品 → 水槍（延長射程）
   - 漢堡排 → 碎岩（打碎更硬的岩石）
   - 麵包 → 居合斬（更大範圍）

招式強化效果持續到你用完強化後的 PP 為止。

## 寶可夢料理助手

烹飪時身邊有特定特長的寶可夢可以改變料理結果：

- **砍伐特長** — 把沙拉變成切絲沙拉（例：飛天螳螂、凱羅斯）
- **粉碎特長** — 把沙拉變成碎莓沙拉（例：赫拉克羅斯）
- **燃燒特長** — 把麵包變成麵包碗（例：小火龍、六尾）

確保你的助手寶可夢正在跟隨你或被放置在料理台附近。

## 口味與寶可夢喜好

每道菜都有口味（甜味、酸味、辣味、苦味、澀味）。你可以把食物送給喜好對應口味的寶可夢，來提升牠們在棲息地的舒適度。在圖鑑中查看每隻寶可夢的喜好來找到牠們偏好的口味。

## 小貼士

- 隨身攜帶食材 — 你隨時可能需要快速恢復 PP
- 把料理台放在主要工作台附近，方便使用
- 多製作幾個篝火 — 4 種烹飪工具中有 3 種需要火源
- 嘗試不同的寶可夢助手來發現新的食譜變化
