# Pokopia 環境等級攻略：提升方法與各等級解鎖內容

環境等級是 Pokémon Pokopia 的核心進度系統。每個城鎮都有獨立的環境等級（1–10），代表該區域對寶可夢的宜居程度。提升等級可解鎖新的商店道具、挑戰任務、寶可夢中心改裝套件，以及推進故事進度。本攻略涵蓋所有你需要知道的內容。

## 什麼是環境等級？

環境等級是按城鎮計算的評級，由該城鎮所有寶可夢朋友的**舒適度**（すみごこち）總和決定。

- 每個城鎮從**等級 1** 開始，最高可達**等級 10**（最大值）
- 等級**按城鎮獨立計算** — 提升一個城鎮不會影響其他城鎮
- 可在寶可夢中心外的 **PC** 或「環境等級詳細資訊」中查看

## 如何提升環境等級

### 1. 結交更多寶可夢朋友

最基本的方法。建造[棲息地](/habitat)吸引新的寶可夢，待牠們出現後與之成為朋友。寶可夢朋友越多 = 總舒適度越高 = 環境等級越高。

- 每個城鎮建議結交 **30–40 隻**寶可夢以達到高效提升 — 在[寶可夢圖鑑](/pokedex)中瀏覽所有寶可夢
- 注意地面上的**閃光亮點** — 這是寶可夢痕跡，能揭示棲息地配方
- 可以建造**重複的棲息地**來吸引更多寶可夢

### 2. 提升寶可夢舒適度

每隻寶可夢都有獨立的舒適度等級，分為六個階段：

| 階段 | 評價 |
|------|------|
| ★★★★★ | 超棒 |
| ★★★★ | 非常好 |
| ★★★ | 不錯 |
| ★★ | 普通 |
| ★ | 微妙 |
| — | 無住所 |

沒有房屋的寶可夢舒適度為**無住所**（等同於 0）。提升方法：

- 在棲息地附近**擺放家具**（放鬆型、玩具型、裝飾型）— 查看[製作配方](/crafting)和[道具一覽](/items)
- **餵食**牠們喜歡的食物（直到滿足）— 參考[料理攻略](/guides/cooking-guide)了解食譜
- **調整環境**符合偏好（溫暖/涼爽、[濕潤/乾燥](/guides/how-to-raise-humidity)、明亮/昏暗）
- 為牠們**建造房屋**（棲息地升級）
- **完成每日請求** — 與寶可夢對話選擇「舒適度如何？」（每天一次）

### 3. 完成寶可夢的請求

有待完成請求的寶可夢**頭上會顯示圖標**。完成請求會直接提升牠們的舒適度。請求每天重置，所以每天都要檢查。

### 4. 修建通往寶可夢中心的道路

寶可夢會前往寶可夢中心社交。從棲息地到中心修建暢通的道路能提升舒適度。填補道路上的坑洞，在高低差處安裝階梯。

### 5. 恢復環境

一般的環境修復也有助於提升：

- **為枯萎的草地、花朵和樹木澆水**使其復甦
- **移除不協調的地形方塊**
- **種植綠色植物**改善景觀

### 6. 善用傳說與幻之寶可夢

傳說和幻之寶可夢對環境等級的**影響比一般寶可夢更大**。優先提升牠們的舒適度可最大化效率。

隨著環境等級提升，新內容會逐步開放：

| 等級 | 解鎖內容 |

- **陰沉沉海邊**（Bleak Beach）

## 環境等級下降的原因

注意 — 以下行為可能**降低**環境等級：

- **[將寶可夢移到其他城鎮](/guides/how-to-move-house)** 會移除該寶可夢的舒適度貢獻（舒適度越高的寶可夢影響越大）
- **損壞的家具**未修理會降低舒適度
- **火焰類物品**（火把、營火等）會在下雨時熄滅 — 請搭建屋頂保護
- 從區域中**移除寶可夢**會降低總分，可能導致已解鎖的內容重新鎖定

## 高效提升技巧

1. **將棲息地集中在寶可夢中心附近** — 一件家具可以惠及多隻寶可夢
2. **等級 3 時改裝寶可夢中心** — 改裝後的中心能促進寶可夢活動
3. **每天檢查舒適度** — 「舒適度如何？」對話能準確顯示每隻寶可夢的需求
4. **優先處理傳說與幻之寶可夢** — 每點舒適度對環境等級的貢獻更大
5. **別忽略道路維護** — 暢通的道路是簡單又有效的舒適度提升方式
6. **保護火焰物品免受雨淋** — 搭建屋頂或遮蔽物防止雨天舒適度下降
