# Pokémon Pokopia 如何搬遷房屋

把房子蓋錯位置了？不用拆掉浪費材料。Pokémon Pokopia 可以使用搬遷套件來移動建築，還能隨時用百變怪旗幟更換你的家。

## 如何搬動小型建築

小型建築可以直接搬起來：

1. 走到建築前面對它
2. 長按 **A 鍵**
3. 角色會舉起建築
4. 搬到想要的位置放下

這只適用於小型的玩家建造物，例如基本的寶可夢小屋。

## 如何用搬遷套件移動大型建築

較大的建築需要使用**搬遷套件**。這是最安全的搬遷方式，不會損失任何材料。

### 步驟：

1. 製作或購買**搬遷套件**
2. 把它放在要搬遷的建築正前方
3. 指派一隻擁有**推平**特長的寶可夢操作套件
4. 寶可夢會準備好建築的搬遷
5. 選擇新的放置位置

> **注意：** 只有玩家建造的建築才能搬遷。地圖上原有的建築（如燈塔）無法移動。

### 擁有推平特長的寶可夢

操作搬遷套件需要擁有**推平**特長的寶可夢。

| 寶可夢 | 屬性 | 特長 |
|--------|------|------|
| 小卡比獸 | 一般 | 推平 |
| 盆才怪 | 岩石 | 推平 |
| 幕下力士 | 格鬥 | 推平、建造 |
| 超力王 | 格鬥 | 推平、建造 |
| 大岩蛇 | 岩石 / 地面 | 推平、粉碎 |
| 大鋼蛇 | 鋼 / 地面 | 推平、粉碎 |
| 幼基拉斯 | 岩石 / 地面 | 推平、粉碎 |
| 沙基拉斯 | 岩石 / 地面 | 推平、粉碎 |
| 班基拉斯 | 岩石 / 惡 | 推平、粉碎 |
| 大顎蟻 | 地面 | 推平、掉落 |
| 超音波幼蟲 | 地面 / 龍 | 推平、飛行 |
| 沙漠蜻蜓 | 地面 / 龍 | 推平、飛行 |

> **提示：** 小卡比獸和盆才怪是最容易在前期找到的推平寶可夢。

## 搬遷套件 vs 拆除套件

| 道具 | 功能 | 會損失材料嗎？ |
|------|------|---------------|
| 搬遷套件 | 把建築移到新位置 | 不會 |
| 拆除套件 | 完全摧毀建築 | 會 |

如果只是想重新擺放建築，請使用搬遷套件。只有在要完全移除建築時才使用拆除套件。

## 如何用百變怪旗幟設定你的家

在 Pokopia 中，你的「家」就是外面插了**百變怪旗幟**的那棟建築。這會決定你**回家**快速旅行的目的地。

### 製作百變怪旗幟

| 材料 | 數量 |
|------|------|
| 堅固樹枝 | 1 |
| 麻繩 | 1 |
| 蘋野果 | 1 |

### 使用方式：

1. 在工作台製作百變怪旗幟
2. 把它放在任何你建造的寶可夢小屋外面
3. 該建築就成為你的指定住所

想換家的話，拿起百變怪旗幟放到另一棟建築外面就好。

## 回家快速旅行

設定好百變怪旗幟後，你可以隨時快速旅行回家：

1. 打開選單
2. 選擇**回家**
3. 你會被傳送到指定的住所

這在遠離基地探索或在偏遠地區收集資源後特別有用。

## 快速提示

1. **搬遷而非拆除** — 使用搬遷套件可以保留所有材料
2. **提前規劃佈局** — 建造前先想好不同棲息地區域的位置
3. **盡早獲得推平寶可夢** — 搬遷套件和地形改造都需要牠
4. **隨擴展移動旗幟** — 把百變怪旗幟放在你最常使用的基地
5. **世界建築無法移動** — 不要嘗試移動燈塔等原有地標
