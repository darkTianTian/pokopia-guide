# Pokopia 环境等级攻略：提升方法与各等级解锁内容

环境等级是 Pokémon Pokopia 的核心进度系统。每个城镇都有独立的环境等级（1–10），代表该区域对宝可梦的宜居程度。提升等级可解锁新的商店道具、挑战任务、宝可梦中心改装套件，以及推进故事进度。本攻略涵盖所有你需要知道的内容。

## 什么是环境等级？

环境等级是按城镇计算的评级，由该城镇所有宝可梦朋友的**舒适度**（すみごこち）总和决定。

- 每个城镇从**等级 1** 开始，最高可达**等级 10**（最大值）
- 等级**按城镇独立计算** — 提升一个城镇不会影响其他城镇
- 可在宝可梦中心外的 **PC** 或「环境等级详细信息」中查看

## 如何提升环境等级

### 1. 结交更多宝可梦朋友

最基本的方法。建造[栖息地](/habitat)吸引新的宝可梦，待它们出现后与之成为朋友。宝可梦朋友越多 = 总舒适度越高 = 环境等级越高。

- 每个城镇建议结交 **30–40 只**宝可梦以达到高效提升 — 在[宝可梦图鉴](/pokedex)中浏览所有宝可梦
- 注意地面上的**闪光亮点** — 这是宝可梦痕迹，能揭示栖息地配方
- 可以建造**重复的栖息地**来吸引更多宝可梦

### 2. 提升宝可梦舒适度

每只宝可梦都有独立的舒适度等级，分为六个阶段：

| 阶段 | 评价 |
|------|------|
| ★★★★★ | 超棒 |
| ★★★★ | 非常好 |
| ★★★ | 不错 |
| ★★ | 普通 |
| ★ | 微妙 |
| — | 无住所 |

没有房屋的宝可梦舒适度为**无住所**（等同于 0）。提升方法：

- 在栖息地附近**摆放家具**（放松型、玩具型、装饰型）— 查看[制作配方](/crafting)和[道具一览](/items)
- **喂食**它们喜欢的食物（直到满足）— 参考[料理攻略](/guides/cooking-guide)了解食谱
- **调整环境**符合偏好（温暖/凉爽、[湿润/干燥](/guides/how-to-raise-humidity)、明亮/昏暗）
- 为它们**建造房屋**（栖息地升级）
- **完成每日请求** — 与宝可梦对话选择「舒适度如何？」（每天一次）

### 3. 完成宝可梦的请求

有待完成请求的宝可梦**头上会显示图标**。完成请求会直接提升它们的舒适度。请求每天重置，所以每天都要检查。

### 4. 修建通往宝可梦中心的道路

宝可梦会前往宝可梦中心社交。从栖息地到中心修建畅通的道路能提升舒适度。填补道路上的坑洞，在高低差处安装阶梯。

### 5. 恢复环境

一般的环境修复也有助于提升：

- **为枯萎的草地、花朵和树木浇水**使其复苏
- **移除不协调的地形方块**
- **种植绿色植物**改善景观

### 6. 善用传说与幻之宝可梦

传说和幻之宝可梦对环境等级的**影响比一般宝可梦更大**。优先提升它们的舒适度可最大化效率。

## 各等级解锁内容

随着环境等级提升，新内容会逐步开放：

| 等级 | 解锁内容 |
|------|---------|
| 2 | 商店与通讯功能；新增可购买道具 |
| 3 | 宝可梦中心改装套件上架商店 |
| 5 | 特殊道具奖励；故事进度里程碑 |
| 5 以上 | 宝可梦生活 PC 新增高难度挑战 |
| 10 | 所有商店道具解锁；宝可梦中心设计套件；配方奖励 |

### 故事推进

要触发游戏结局，需要在以下四个区域达到**环境等级 5**：

- **干巴巴荒野**（Withered Wasteland）
- **阴沉沉海边**（Bleak Beach）
- **凸隆隆山地**（Rocky Ridges）
- **亮晶晶空岛**（Sparkling Skylands）

## 环境等级下降的原因

注意 — 以下行为可能**降低**环境等级：

- **[将宝可梦移到其他城镇](/guides/how-to-move-house)** 会移除该宝可梦的舒适度贡献（舒适度越高的宝可梦影响越大）
- **损坏的家具**未修理会降低舒适度
- **火焰类物品**（火把、营火等）会在下雨时熄灭 — 请搭建屋顶保护
- 从区域中**移除宝可梦**会降低总分，可能导致已解锁的内容重新锁定

## 高效提升技巧

1. **将栖息地集中在宝可梦中心附近** — 一件家具可以惠及多只宝可梦
2. **等级 3 时改装宝可梦中心** — 改装后的中心能促进宝可梦活动
3. **每天检查舒适度** — 「舒适度如何？」对话能准确显示每只宝可梦的需求
4. **优先处理传说与幻之宝可梦** — 每点舒适度对环境等级的贡献更大
5. **别忽略道路维护** — 畅通的道路是简单又有效的舒适度提升方式
6. **保护火焰物品免受雨淋** — 搭建屋顶或遮蔽物防止雨天舒适度下降
