# ハネッコの綿胞子

![ハネッコイベントバナー](/images/events/hoppip-event-hero.png)

Pokémon Pokopia 初の期間限定イベント **「More Spores for Hoppip（ハネッコの綿胞子）」** が **2026年3月9日～3月24日（太平洋時間）** に開催中です。このイベント期間中のみ、ハネッコ・ポポッコ・ワタッコの3匹と友達になることができます。

> Pokémon GO のタイムリサーチ（Timed Research）とは異なり、Pokopia のイベントは綿胞子の収集と特定の生息地づくりで進行します。以下の完全ガイドをご覧ください。

## イベント情報

| 項目 | 詳細 |
|------|------|
| イベント名 | More Spores for Hoppip |
| 開始 | 2026年3月9日 午後1時（太平洋時間） |
| 終了 | 2026年3月24日 午後12時59分（太平洋時間） |
| 参加条件 | 町にポケモンセンターを再建済みであること |
| Cloud Island | 対象外 |

## ステップバイステップガイド

### ステップ1：ポケモンセンターでハネッコと話す

イベント開始後、**ポケモンセンター**（イベント中は花で飾られています）に向かいましょう。近くにハネッコがいるので話しかけると、**綿胞子**を集めてほしいと頼まれます。

### ステップ2：ドリームアイランドで綿胞子を集める

綿胞子はイベント期間中の**ドリームアイランド**にのみ出現します。ドリームアイランドに行くには、**フワンテ（Drifloon）**に人形を渡す必要があります。訪問のたびにできるだけ多くの胞子を集めましょう。

### ステップ3：胞子をピクニック家具と交換する

**ポケモンセンター**に戻り、綿胞子を**ピクニックテーマの家具アイテム**と交換します。これらが次のステップで生息地を作るための材料になります。

### ステップ4：生息地を作ってポポッコ・ワタッコを引き寄せる

家具を使って特定の生息地を作りましょう：

| 生息地 | 必要アイテム | 引き寄せるポケモン |
|--------|------------|------------------|
| フィールドトリップフレンズ | フラワーバックパック、お弁当箱、ハネッコ水筒（2×2） | ポポッコ |
| タンポポランチタイム | タンポポの花、フラワークッション、花のテーブルセット、お弁当箱（2×2） | ワタッコ |
| イエローカーペット | 4× タンポポの花（2×2） | ハネッコ・ポポッコ・ワタッコ |

> この3種のポケモンは**イベント期間中のみ出現**します。コレクターの方はぜひ参加しましょう。Pokopia ではポケモンは進化しません — ハネッコ・ポポッコ・ワタッコはそれぞれ専用の生息地で個別に友達になる必要があります。

## 注意事項

- このイベントは**Cloud Islandでは利用不可**
- イベント内容にアクセスするには約**30分のプレイ時間**が必要（ポケモンセンターの再建が必要）
- 早期購入特典の**メタモンラグ**は引き続き「ふしぎなおくりもの」から受け取れます（2027年1月31日まで）
