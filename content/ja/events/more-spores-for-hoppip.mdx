# ハネッコの綿胞子

![ハネッコイベントバナー](/images/events/hoppip-event-hero.png)

Pokémon Pokopia 初の期間限定イベント **「More Spores for Hoppip（ハネッコの綿胞子）」** が **2026年3月9日～3月24日（太平洋時間）** に開催中です。このイベント期間中のみ、ハネッコ・ポポッコ・ワタッコの3匹と友達になることができます。

## イベント情報

| 項目 | 詳細 |
|------|------|
| イベント名 | More Spores for Hoppip |
| 開始 | 2026年3月9日 午後1時（太平洋時間） |
| 終了 | 2026年3月24日 午後12時59分（太平洋時間） |
| 参加条件 | 町にポケモンセンターを再建済みであること |
| Cloud Island | 対象外 |

## イベント内容

### ハネッコ・ポポッコ・ワタッコと友達になる

![ハネッコイベントゲームプレイスクリーンショット](/images/events/hoppip-event-screenshot.png)

イベント期間中、ハネッコがあなたの町に出現します。**ピクニックテーマの生息地**を作ることで、ポポッコとワタッコも引き寄せることができます。

> この3種のポケモンは**イベント期間中のみ出現**します。コレクターの方はぜひ参加しましょう。

### 綿胞子を集める

イベント期間中、町に特別な**綿胞子**が出現します。集めた胞子は**ピクニックテーマの家具アイテム**と交換でき、生息地の装飾に使えます。

## 参加方法

1. 町に**ポケモンセンター**を再建していることを確認（約30分のプレイが必要）
2. イベント開始後、町に出現するハネッコを探す
3. 生息地に散らばっている綿胞子を集める
4. 胞子をピクニックテーマの家具と交換する
5. ピクニックテーマの生息地を作ってポポッコとワタッコを引き寄せる

## 注意事項

- このイベントは**Cloud Islandでは利用不可**
- イベント内容にアクセスするには約**30分のプレイ時間**が必要
- 早期購入特典の**メタモンラグ**は引き続き「ふしぎなおくりもの」から受け取れます（2027年1月31日まで）
