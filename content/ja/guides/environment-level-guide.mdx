# Pokopia 環境レベル攻略：効率的な上げ方とレベル別解放要素

環境レベルは Pokémon Pokopia の中心的な進行システムです。各街にはそれぞれ独立した環境レベル（1〜10）があり、ポケモンにとっての住みやすさを示しています。レベルを上げるとショップのアイテム、チャレンジ、ポケモンセンター改装キットなどが解放されます。このガイドでは必要な情報をすべて解説します。

## 環境レベルとは？

環境レベルは街ごとの評価で、その街に住むすべてのポケモンの「**すみごこち**」の合計値で決まります。

- 各街は**レベル1**から始まり、**レベル10**（最大）まで上げられます
- レベルは**街ごとに独立** — ある街のレベルを上げても他の街には影響しません
- ポケモンセンター外の **PC** または「環境レベルのくわしい情報」で確認できます

## 環境レベルの上げ方

### 1. ポケモンをたくさん友達にする

最も基本的な方法です。[生息地](/habitat)を作って新たなポケモンを呼び寄せ、友達になりましょう。友達のポケモンが多いほど、合計すみごこちが上がり、環境レベルも上がります。

- 効率的なレベル上げには、1つの街で**30〜40匹**のポケモンを目指しましょう — [ポケモン図鑑](/pokedex)で全ポケモンを確認できます
- 地面の**光るキラキラ**を探しましょう — ポケモンの痕跡で、生息地のレシピがわかります
- **同じ生息地の複製**を作ることで、追加のポケモンを呼べます

### 2. ポケモンのすみごこちを上げる

各ポケモンには個別のすみごこちレベルがあり、6段階で評価されます：

| 段階 | 評価 |
|------|------|
| ★★★★★ | さいこう |
| ★★★★ | とってもいい |
| ★★★ | いいかんじ |
| ★★ | ふつう |
| ★ | びみょう |
| — | おうちなし |

おうちのないポケモンのすみごこちは**おうちなし**（実質0）です。上げるには：

- 生息地の近くに**家具を置く**（リラックス・おもちゃ・デコレーション系）— [クラフトレシピ](/crafting)と[アイテム一覧](/items)を確認
- 好きな**ごはんをあげる**（満足するまで）— [料理ガイド](/guides/cooking-guide)でレシピをチェック
- **環境を好みに合わせる**（暖かさ・[湿度](/guides/how-to-raise-humidity)・明るさなど）
- **おうちを建てる**（生息地のアップグレード）
- **毎日のお願いをこなす** — ポケモンに話しかけて「すみごこち どう？」を選択（1日1回）

### 3. ポケモンのおねがいごとを達成する

おねがいごとがあるポケモンは**頭の上にアイコン**が表示されます。おねがいを達成するとすみごこちが直接上がります。おねがいは毎日リセットされるので、毎日チェックしましょう。

### 4. ポケモンセンターへの道を整備する

ポケモンはポケモンセンターを訪れて交流します。生息地からセンターまでの道を整備すると、すみごこちが向上します。道路の穴をふさぎ、段差には階段を設置しましょう。

### 5. 環境を復元する

一般的な環境復元もカウントされます：

- **枯れた草・花・木に水をやって**復活させる
- **不自然な地形ブロック**を除去する
- **緑を植えて**景観を改善する

### 6. 伝説・幻のポケモンを活用する

伝説・幻のポケモンは通常のポケモンより**環境レベルへの影響が大きい**です。効率を最大化するなら、彼らのすみごこちを優先的に上げましょう。

## レベル別の解放要素

環境レベルが上がるにつれて、新しいコンテンツが利用可能になります：

| レベル | 解放内容 |
|--------|---------|
| 2 | ショップ・通信機能；新規購入アイテム追加 |
| 3 | ポケモンセンター改装キットがショップに登場 |
| 5 | 特別アイテム報酬；ストーリー進行のマイルストーン |
| 5以上 | ポケライフPCに新チャレンジ追加（難易度上昇） |
| 10 | すべてのショップアイテム解放；ポケモンセンターデザインキット；レシピ報酬 |

### ストーリー進行

エンディングを見るには、以下の4つのエリアで**環境レベル5**を達成する必要があります：

- **パサパサこうや**（Withered Wasteland）
- **ドンヨリうみべ**（Bleak Beach）
- **ゴツゴツやま**（Rocky Ridges）
- **キラキラうきしま**（Sparkling Skylands）

## 環境レベルが下がる原因

注意 — 以下の行動で環境レベルが**下がる**可能性があります：

- **[ポケモンを別の街に移動](/guides/how-to-move-house)**させると、そのポケモンのすみごこち分が減少（すみごこちが高いほど影響大）
- **壊れた家具**を修理せずに放置するとすみごこちが低下
- **火を使うオブジェクト**（たいまつ・焚き火など）は雨で消える — 屋根で保護しましょう
- エリアから**ポケモンを除外**すると合計スコアが下がり、解放済みコンテンツがロックされる場合あり

## 効率的なレベル上げのコツ

1. **生息地をポケモンセンター付近にまとめる** — 1つの家具で複数のポケモンに効果を発揮できます
2. **レベル3でポケモンセンターを改装** — 改装後のセンターはポケモンの活動を促進します
3. **毎日すみごこちを確認** — 「すみごこち どう？」で各ポケモンが何を求めているか正確にわかります
4. **伝説・幻のポケモンを優先** — すみごこち1ポイントあたりの環境レベルへの貢献が大きいです
5. **道の整備を怠らない** — 簡単なすみごこちアップで、エリア内の全ポケモンに効果があります
6. **火のオブジェクトを雨から守る** — 屋根やシェルターを作って、雨天時のすみごこち低下を防ぎましょう
