# Pokémon Pokopia 全素材ガイド

Pokémon Pokopia に登場するすべてのクラフト素材を網羅したガイドです。木材、レンガ、紙、コンクリートなど、どこで見つけてどう加工するかを詳しく解説します。

## 素材の仕組み

Pokopia の素材は2種類に分かれます：

- **原素材** — 地面で拾う、オブジェクトを壊して入手、または「ちらかす」とくぎのポケモンが落とす
- **加工素材** — 原素材を特定のとくぎポケモンに渡して作る（きる、もやす、くだく、リサイクル）

> **ヒント：** 収納ボックスをクラフト台の近くに置くと、中の素材を直接使えます。

## 原素材

### 石

最も一般的な素材。探索中に拾うか、いわくだきで岩を壊して入手。全エリアにあります。

### 葉っぱ

探索中に拾うか、いあいぎりで草やツルを切って入手。フシギバナなど「ちらかす」とくぎのポケモンも落とします。

### 小さな丸太

木材の原料。入手方法：

- **いあいぎり** で木や朽ちた木材を切る
- ポケモンからのプレゼント
- 海岸の水面キラキラスポットを調べる

### 丈夫な枝

探索中に拾うか、いあいぎりで木製オブジェクトを切る。ノクタスも落とします。

### ツルロープ

**いあいぎり** でツルを切る、または「ちらかす」とくぎのポケモン（**マダツボミ**、**ウツドン**）が生息地付近に落とします。

### ぐにゃぐにゃ粘土

**いわくだき** や **ころがる** で粘土ブロックを壊して入手。**ブリーク・ビーチ** に大量にあります。じめんタイプの「ちらかす」ポケモン（**パルデアウパー**）も落とします。

### ひかるキノコ

洞窟内で見つかります。枯れた荒野やその他の洞窟の暗い場所を探しましょう。

### ハチミツ

キラキラ光る花壇を調べるか、「みつあつめ」とくぎのポケモン（**ミツハニー**）が生産します。

### 草原の花

各エリアの花畑で拾えます。

### フワフワ

「ちらかす」とくぎの **メリープ**、**モココ** が生息地に落とします。柔らかい家具の素材に使います。

### ひも

布ゴミを分解、クモの巣を切る、またはむしタイプの「ちらかす」ポケモン（**イトマル**）が落とします。

### わた

フワフワと同様、探索中に入手またはポケモンから取得。

### いとくず

探索中に拾うか、布製品を分解して入手。

### 汚れた紙

**キラキラ・スカイランド** で拾えます。きれいな紙の原料です。

### 燃えないゴミ

**ブリーク・ビーチ** で拾えます。「リサイクル」とくぎのポケモンで鉄鉱石に変換可能。

### シーグラスの欠片

ブリーク・ビーチで拾えます。ベトベトンに覆われたタイル上に多い。

### 貝がら

**ブリーク・ビーチ** の海岸線で拾えます。

### 火山灰

火山エリアで見つかります。溶鉱炉でガラスに加工できます。

### 石灰石

**キラキラ・スカイランド** に何千ブロックも存在。コンクリートの原料。

### 銅鉱石

**ロッキーリッジ** の鉱山で鉱石ブロックを壊して入手。

### 鉄鉱石

**ロッキーリッジ** の鉱山で鉱石ブロックを壊すか、燃えないゴミを「リサイクル」とくぎのポケモンに渡す。

### 金鉱石

**ロッキーリッジ** の鉱山で鉱石ブロックを壊して入手。銅や鉄より希少。

### ポケメタルの欠片

枯れた荒野とキラキラ・スカイランドで鉱石ブロックを壊して入手。

### クリスタルの欠片

**ロッキーリッジ** の鉱山で大きなクリスタルを壊して入手。

## 小麦の入手方法

小麦はチーム入会チャレンジや各種料理に必要です：

1. **最初の場所：** ロッキーリッジ — 荒れた農場に乾燥した小麦があり、水をやると収穫可能
2. **小麦の種：** ロッキーリッジのポケモンからもらうか、赤いモンスターボールや水のキラキラから入手
3. **小麦の栽培：** **たがやす** 技で畑を作り、種をまいて水をやる。「そだてる」とくぎのポケモンで成長を加速
4. **小麦の購入：** ロッキーリッジの環境レベルが3になると購入可能

> **注意：** ロッキーリッジにはチームが「グレート」ランクに達する必要があります。

## 加工素材

特定のとくぎを持つポケモンが必要です。

### 木材の入手方法

**必要なとくぎ：** きる（ストライク、カイロス、ヘラクロス）

**小さな丸太** を「きる」とくぎのポケモンに渡します。「これを見て」を選んで丸太を渡しましょう。

**変換レート：** 小さな丸太10個 → 木材50個

> 詳しくは [木材の入手方法ガイド](/ja/guides/how-to-get-lumber) をご覧ください。

### レンガの入手方法

**必要なとくぎ：** もやす（アチャモ、ブースター、ウインディ、キュウコン）

**ぐにゃぐにゃ粘土** を「もやす」とくぎのポケモンに渡します。

**変換レート：** ぐにゃぐにゃ粘土10個 → レンガ20個

**粘土の入手先：** ブリーク・ビーチでいわくだきやころがるで粘土ブロックを壊す。パルデアウパー（ちらかす）も落とします。

### 紙の入手方法

**必要なとくぎ：** リサイクル（ヤブクロン、ダストダス、ポリゴン、チラチーノ）

**汚れた紙** を「リサイクル」とくぎのポケモンに渡します。汚れた紙はキラキラ・スカイランドで拾えます。

### コンクリートの入手方法

**必要なとくぎ：** くだく（ローブシン、イワーク、メタグロス）

まず **コンクリートミキサー**（鉄のインゴット2個で制作）が必要。水の近くに設置し、**石灰石** を入れ、「くだく」とくぎのポケモンに加工させます。

**石灰石の入手先：** キラキラ・スカイランドに何千ブロックもあります。

### 鉄のインゴットの入手方法

**必要：** 溶鉱炉 + もやすとくぎポケモン

溶鉱炉で **鉄鉱石** を溶かします。もやすとくぎのポケモンが炉に火を入れます。

### 銅のインゴットの入手方法

**必要：** 溶鉱炉

溶鉱炉で **銅鉱石** を溶かします。鉄のインゴットと同じ手順です。

### 金のインゴットの入手方法

**必要：** 溶鉱炉

溶鉱炉で **金鉱石** を溶かします。金鉱石はより希少で、ロッキーリッジの奥で見つかります。

### ポケメタルの入手方法

**必要：** 溶鉱炉

溶鉱炉で **ポケメタルの欠片** を溶かします。ビッグ収納ボックスなど高レベルアイテムに使用。

### ガラスの入手方法

**必要：** 溶鉱炉 + もやすとくぎ

溶鉱炉で **火山灰**、**砂**、**砂岩** を溶かします。

### 絵の具の入手方法

**必要なとくぎ：** くだく（イワーク、ローブシン、パーモット）

**ヒメリのみ** などのきのみを「くだく」とくぎのポケモンに渡します。きのみの種類で色が変わります。

## ポケモンのとくぎと素材の対応表

| とくぎ | 機能 | 代表ポケモン |
|--------|------|-------------|
| **きる** | 小さな丸太 → 木材 | ストライク、カイロス、ヘラクロス |
| **もやす** | 粘土 → レンガ、鉱石の精錬、ガラス制作 | アチャモ、ブースター、ウインディ、キュウコン |
| **くだく** | 石灰石 → コンクリート、きのみ → 絵の具 | イワーク、ローブシン、メタグロス、パーモット |
| **リサイクル** | 汚れた紙 → 紙、ゴミ → 鉄鉱石 | ヤブクロン、ダストダス、ポリゴン、チラチーノ |
| **ちらかす** | 生息地付近に原素材を落とす | メリープ（フワフワ）、イトマル（ひも）、マダツボミ（ツルロープ）、パルデアウパー（粘土） |
| **そだてる** | 作物の成長を加速 | マダツボミ、ジャノビー、クサイハナ |
| **あつめる** | コミュニティボックスに自動収集 | 各種ポケモン |

## 素材を効率よく集めるコツ

1. **「ちらかす」ポケモンの生息地を早めに作る** — メリープ、イトマル、マダツボミが自動でフワフワ、ひも、ツルロープを生産
2. **素材は満タンで渡す** — 一度に10個渡して効率最大化
3. **加工中もこまめにチェック** — 途中でも完成分を受け取れる
4. **ドリームアイランドを探索** — 素材、ブロック、レシピが集まる専用リソースエリア
5. **収納ボックスをクラフト台の横に** — 中の素材を直接使える
6. **「あつめる」とくぎのポケモンを活用** — コミュニティボックスに自動収集で手間を省く
