# ぽこ あ ポケモン 予約ガイド

『ぽこ あ ポケモン』（英語：Pokémon Pokopia）は、ポケモンシリーズ初のスローライフサンドボックスゲームです。**2026年3月5日**に世界同時発売、対応プラットフォームは **Nintendo Switch 2** です。

本記事では、各地域の購入先、価格、予約特典をまとめました。

## ゲーム基本情報

- **タイトル**：ぽこ あ ポケモン / Pokémon Pokopia
- **発売日**：2026年3月5日（世界同時発売）
- **対応機種**：Nintendo Switch 2
- **ジャンル**：スローライフサンドボックス
- **プレイ人数**：1～4人（オンラインマルチ対応）
- **パッケージ版について**：パッケージ版は「キーカード」方式で、初回プレイ時にインターネット接続とダウンロードが必要（空き容量10GB）

## 各地域の価格

| 地域 | 価格 |
|------|------|
| 日本 | ¥8,980（税込） |
| アメリカ | $69.99 |
| イギリス | £49.99 |

> ダウンロード版とパッケージ版は同じ価格です。ダウンロード版は3月5日0時から即プレイ可能。配送を待つ必要がありません。

## 共通の早期購入特典

すべての購入先で、ゲーム内アイテム **「メタモンラグ」** がもらえます。ゲーム内の「ふしぎなおくりもの」から受け取れます。受け取り期限は **2027年1月31日** まで。

このアイテムは通常プレイでも入手可能なので、期限を逃しても心配ありません。
