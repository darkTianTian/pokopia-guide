# Pokopia コケカビゴン攻略：お供え物のバフ効果とおすすめ料理

コケカビゴン（Mosslax）は、街で眠っている苔に覆われた[カビゴン](/pokedex/snorlax)です。目覚めさせると、毎日食べ物をお供えして翌朝5:00まで続く強力なバフ効果を得ることができます。コケカビゴンは ![icon:Eat](/images/specialties/eat.png) **くいしんぼ**のとくぎを持っています——このガイドでは、全味のバフ効果、おすすめのお供え物、効果を最大化する方法を解説します。

## コケカビゴンのお供えを解放する方法

1. **どんよりうみべ**のストーリーを進める
2. クエスト**「街を明るくしよう！」**をクリア
3. コケカビゴンが目覚め、前に**お供え台**が出現
4. 毎日お供え台に食べ物を置くとバフを獲得

## お供え物のバフ効果一覧

得られるバフはお供え物の**味**によって決まります。全6種類：

| 味 | 効果 | おすすめ場面 |
|----|------|-------------|
| **普通** | ポケモンと仲良くなりやすい | 新しいポケモンとの友情 |
| **からい** | [生息地](/habitat)にポケモンが出現しやすい | [図鑑](/pokedex)完成、環境レベル上げ |
| **しぶい** | 珍しいハネを持つポケモンに出会いやすい（[ホウオウ](/pokedex/ho-oh)・[ルギア](/pokedex/lugia)） | ![icon:にじいろのはね](/images/items/item-31.png) にじいろのハネ・![icon:ぎんいろのはね](/images/items/item-203.png) ぎんいろのハネの入手 |
| **あまい** | いにしえの遺物や落とし物が見つかりやすい | 化石集め、![icon:ナゾのせきばん](/images/items/item-226.png) ナゾのせきばん（[ミュウ](/pokedex/mew)用） |
| **にがい** | 探索で珍しいものが見つかりやすい | 素材集め |
| **すっぱい** | ショップにいいものが並びやすい | レアアイテムの購入 |

## おすすめのお供え物

### 伝説ポケモン狙い：しぶい味

![icon:きのこスープ](/images/cooking/recipes/mushroom-soup.png) **きのこスープ**や ![icon:きのこハンバーグ](/images/cooking/recipes/mushroom-hamburger.png) **きのこハンバーグ**（しぶい味）をお供えすると、[ホウオウ](/pokedex/ho-oh)や[ルギア](/pokedex/lugia)の出現確率がアップ。![icon:とうめいなスズ](/images/items/item-372.png) とうめいなスズ・![icon:うみなりのスズ](/images/items/item-355.png) うみなりのスズ作成に必要なハネを集めよう。

### 図鑑完成：からい味

![icon:やくぜんスープ](/images/cooking/recipes/herbal-soup.png) **やくぜんスープ**や ![icon:クルトンサラダ](/images/cooking/recipes/crouton-salad.png) **クルトンサラダ**（からい味）で[生息地](/habitat)のポケモン出現率アップ。多くのプレイヤーにおすすめの日常バフ。

### ミュウ探し：あまい味

![icon:ポテトハンバーグ](/images/cooking/recipes/potato-hamburger.png) **ポテトハンバーグ**（あまい味）をお供えすると、パサパサこうやでの ![icon:ナゾのせきばん](/images/items/item-226.png) ナゾのせきばんのドロップ率がアップ。全27枚集めると[ミュウ](/pokedex/mew)が解放される。

## 生食 vs 料理：違いはある？

**あります！** 料理済みの食べ物は生の食材より**強いバフ効果**が得られます。必ず[料理](/cooking)してからお供えしましょう。

例：
- 生の食材をお供え → **基本**バフ
- 同じ味の料理をお供え → **強化**バフ

[料理レシピ](/cooking)ページで味ごとの全レシピを確認できます。

## 重要ルール

- **1日1回** — ゲーム内の1日に1回だけお供え可能
- **翌朝5:00まで持続** — 一度発動すると1日中効果が続く
- **変更不可** — お供えしたらその日はバフを変えられない
- **途中変更しない** — 味を決めたらその日はそのまま

## 毎日のおすすめルーティン

1. **目標を確認** — 今日はハネが必要？ポケモン出現？レアアイテム？
2. **対応する料理を作る** — 目標に合った味を選ぶ（[料理レシピ](/cooking)を参照）
3. **コケカビゴンにお供え** — 料理をお供え台に置く
4. **1日中バフを活用** — 効果は明日の朝5:00まで続く

> **Tip：** コケカビゴンのバフと[ゆめのしま](/guides/dream-island-guide)探索を組み合わせると、毎日の効率が最大化。ハネ集め時にしぶい味バフを使い、ゆめのしまで追加の素材を入手しよう。
