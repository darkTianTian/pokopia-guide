# Pokopia ゆめしま攻略：全人形・島の素材・伝説ポケモン

ゆめしま（Dream Island）は Pokémon Pokopia の特別な無人島で、レア素材、ほしのすな、さらには伝説ポケモンが見つかります。人形をフワンテに見せると、毎日異なる島を訪れて素材を集められます。このガイドでは全人形の入手場所、島の素材、攻略テクニックを解説します。

## ゆめしまの仕組み

1. 探索中に**ポケモンの人形**を見つける
2. 人形を**フワンテ**に見せる（または近くで調べる）
3. フワンテが対応する**ゆめしま**に連れて行ってくれる
4. 島を探索、素材を集め、地下洞窟で伝説ポケモンをチェック
5. 集めた素材を持って帰還

**毎日のルール：**
- 1日に行けるゆめしまは **1箇所のみ**
- 素材は**毎日リセット** — 翌日また訪問すれば再収集可能
- **朝5:00** にリセット
- 同日中に同じ島を再訪問可能ですが、素材は翌日までリスポーンしません

## ゆめしまの伝説ポケモン

各ゆめしま（いわやま島を除く）では、**地下洞窟**に伝説ポケモンが出現する可能性があります。到着後、中央エリア付近の **2つの洞窟入口** を探しましょう。

| 島 | 伝説ポケモン |
|----|-------------|
| こうや島 | [スイクン](/pokedex/suicune) |
| うなばら島 | [ライコウ](/pokedex/raikou) |
| かざん島 | [エンテイ](/pokedex/entei) |
| そら島 | [ミュウツー](/pokedex/mewtwo) |

> **重要：** [ライコウ](/pokedex/raikou) と [ミュウツー](/pokedex/mewtwo) は、メインストーリーで先に出会っている必要がある場合があります。出現しない場合は、ストーリーを先に進めましょう。

**伝説が出現しなかった場合：** フワンテに戻り、翌日再挑戦。伝説の出現率は低いため、根気よく通いましょう。

## ほしのすなとほしのかけら

全てのゆめしまの海岸線や道沿いに**ほしのすな**が落ちています。**99個のほしのすな** を集めると **1個のほしのかけら** をクラフトでき、マルチプレイの贈り物として非常に価値があります。

> **ヒント：** ほしのすなは見つけたら必ず拾いましょう。コツコツ集めてほしのかけらを作ると、マルチプレイで最高のプレゼントになります。

## ゆめしまで見つかるその他のもの

- **CD** — 収集可能な音楽トラック
- **ノート** — Pokopia の世界の隠された物語
- **すみかヒント** — まだ見つけていないすみかの建設ヒント
- **衣装** — キャラクターテーマの服（グリーン、リーフ、ヒビキ）

## ゆめしま周回のコツ

1. **毎日行く** — 毎日の訪問がレア素材を安定的に集める最良の方法
2. **必要に応じて島を選ぶ** — [コンクリート](/guides/all-crafting-materials)が必要？いわやま島で石灰石を。[ひも](/guides/all-crafting-materials)が必要？うなばら島へ
3. **地下洞窟は必ずチェック** — 伝説ポケモンは洞窟にしか出現しません
4. **ほしのすなを集める** — 99個 → 1個のほしのかけら、マルチプレイ必須
5. **7つの人形を全部集める** — 人形が多いほど、毎日選べる島が増えます
6. **さがすポケモンを活用** — 埋まった人形や隠しアイテムの発見に役立ちます
