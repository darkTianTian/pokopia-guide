# Pokopia 料理ガイド：全レシピ・道具・効果まとめ

料理は Pokémon Pokopia で最も便利なシステムの一つです。食べ物を食べるとPPが全回復し、特定のわざが強化されるため、探索や生息地建設に欠かせません。このガイドでは料理の解放方法から全レシピとその効果まで解説します。

## 料理の解放方法

料理は**ゴツゴツやま**のストーリー（3番目の島）で解放されます。樽に閉じ込められた特別なヨクバリス「シェフ・デンテ」と出会い、救出すると料理の作り方を教えてもらえます。以降、どの料理台でも調理ができるようになります。

## 調理器具

クラフト台で4種類の調理器具を作ります。それぞれ異なる料理に使います：

| 道具 | 素材 | 料理 |
|------|------|------|
| まないた | ざいもく x2 | サラダ |
| なべ | どうののべぼう x2 | スープ |
| フライパン | どうののべぼう x2 | ハンバーグ |
| パンがま | かざんばい x4、いし x2、ちいさなまるた x2 | パン |

**重要：** なべ、フライパン、パンがまは火がついた焚き火やコンロが必要です。たきびの近くに設置しましょう。

## 全レシピ

### サラダ（はっぱカッターを強化）

サラダは最初に覚える料理です。**はっぱカッター**のわざを強化し、ツル、コケ、植物を育てられるようになります。

| レシピ | 材料 | 味 |
|--------|------|-----|
| シンプルサラダ | はっぱ x1 + 他の材料 x1 | — |
| 海藻サラダ | はっぱ x1 + 海藻 x1 | にがい |
| クルトンサラダ | はっぱ x1 + パン x1 | からい |
| ヒメリサラダ | はっぱ x1 + ヒメリのみ x1 | あまい |
| 千切りサラダ | はっぱ x1 + 材料 x1 + きる特技ポケモン | すっぱい |
| つぶしベリーサラダ | はっぱ x1 + カゴのみ x1 + くだく特技ポケモン | しぶい |

### スープ（みずでっぽうを強化）

スープは**みずでっぽう**を強化し、射程と水量が増加します。

| レシピ | 材料 | 味 |
|--------|------|-----|
| シンプルスープ | まひるの水 x1 + 他の材料 x2 | — |
| 海藻スープ | まひるの水 x1 + 海藻 x1 + 他 x1 | にがい |
| ヘルシースープ | まひるの水 x1 + 豆 x1 + はっぱ x1 | からい |
| キノコスープ | まひるの水 x1 + キノコ x1 + 他 x1 | しぶい |

### ハンバーグ（いわくだきを強化）

ハンバーグは**いわくだき**を強化し、より硬い岩や素材を砕けるようになります。

| レシピ | 材料 | 味 |
|--------|------|-----|
| シンプルハンバーグ | 豆 x1 + 他の材料 x3 | — |
| トマトハンバーグ | 豆 x1 + トマト x1 + 他 x2 | すっぱい |
| ポテトハンバーグ | 豆 x1 + ポテト x1 + 他 x2 | あまい |
| キノコハンバーグ | 豆 x1 + キノコ x1 + 他 x2 | しぶい |

### パン（いあいぎりを強化）

パンは**いあいぎり**を強化し、範囲と効果がアップします。

| レシピ | 材料 | 味 |
|--------|------|-----|
| シンプルパン | こむぎ x1 + 他の材料 x2 | — |
| にんじんパン | こむぎ x1 + にんじん x1 + 他 x1 | からい |
| ヒメリパン | こむぎ x1 + ヒメリのみ x1 + 他 x1 | すっぱい |
| パンボウル | こむぎ x1 + スープ x1 + 他 x1 + もやす特技ポケモン | からい |

## 料理の効果

食事には2つの効果があります：

1. **PPが全回復** — 黄色のPPゲージが完全に回復
2. **特定のわざが強化** — 料理の種類によって異なるわざが強化：
   - サラダ → はっぱカッター（植物を育てる）
   - スープ → みずでっぽう（射程延長）
   - ハンバーグ → いわくだき（硬い岩を砕く）
   - パン → いあいぎり（範囲拡大）

わざ強化は強化されたPPを使い切るまで持続します。

## ポケモンの料理サポート

調理中に特定のとくぎを持つポケモンがそばにいると、料理の結果が変わります：

- **きる特技** — サラダが千切りサラダに変化（例：ストライク、カイロス）
- **くだく特技** — サラダがつぶしベリーサラダに変化（例：ヘラクロス）
- **もやす特技** — パンがパンボウルに変化（例：ヒトカゲ、ロコン）

サポートポケモンが自分についてくるか、料理台の近くにいることを確認しましょう。

## 味とポケモンのお気に入り

各料理には味があります（あまい、すっぱい、からい、にがい、しぶい）。ポケモンの好みの味に合った食べ物をプレゼントすると、生息地での快適度がアップします。各ポケモンの好みは図鑑のお気に入りで確認できます。

## 攻略のコツ

- 食材を常に持ち歩こう — いつPP回復が必要になるかわかりません
- 料理台はメインのクラフト台の近くに設置すると便利
- たきびを複数作ろう — 4つの調理器具のうち3つは火が必要
- いろんなポケモンをサポートに試して新しいレシピバリエーションを発見しよう
