# Pokopia 환경 레벨 공략: 효율적으로 올리는 방법과 레벨별 해금 요소

환경 레벨은 Pokémon Pokopia의 핵심 진행 시스템입니다. 각 마을에는 독립적인 환경 레벨(1~10)이 있으며, 해당 지역이 포켓몬에게 얼마나 살기 좋은지를 나타냅니다. 레벨을 올리면 새로운 상점 아이템, 챌린지, 포켓몬센터 개장 키트 등이 해금됩니다. 이 가이드에서 필요한 모든 정보를 확인하세요.

## 환경 레벨이란?

환경 레벨은 마을별 평가 시스템으로, 해당 마을에 사는 모든 포켓몬 친구의 **안락도**(すみごこち) 합계로 결정됩니다.

- 각 마을은 **레벨 1**에서 시작하여 **레벨 10**(최대)까지 올릴 수 있습니다
- 레벨은 **마을별 독립** — 한 마을의 레벨을 올려도 다른 마을에는 영향 없습니다
- 포켓몬센터 앞 **PC** 또는 「환경 레벨 상세 정보」에서 확인 가능합니다

## 환경 레벨 올리는 방법

### 1. 포켓몬 친구 늘리기

가장 기본적인 방법입니다. [서식지](/habitat)를 만들어 새로운 포켓몬을 유치하고, 나타나면 친구가 되세요. 포켓몬 친구가 많을수록 = 총 안락도 상승 = 환경 레벨 상승.

- 효율적인 레벨업을 위해 마을당 **30~40마리** 포켓몬을 목표로 하세요 — [포켓몬 도감](/pokedex)에서 전체 포켓몬 확인
- 바닥의 **반짝이는 빛**을 찾으세요 — 포켓몬 흔적으로 서식지 레시피를 알 수 있습니다
- **같은 서식지를 복제**하여 추가 포켓몬을 유치할 수 있습니다

### 2. 포켓몬 안락도 올리기

각 포켓몬은 개별 안락도 레벨이 있으며 6단계로 평가됩니다:

| 단계 | 평가 |
|------|------|
| ★★★★★ | 최고 |
| ★★★★ | 매우 좋음 |
| ★★★ | 좋음 |
| ★★ | 보통 |
| ★ | 미묘 |
| — | 집 없음 |

집이 없는 포켓몬의 안락도는 **집 없음** (사실상 0)입니다. 올리는 방법:

- 서식지 근처에 **가구 배치** (휴식용·장난감·장식용) — [제작 레시피](/crafting)와 [아이템 목록](/items) 확인
- 좋아하는 **음식 주기** (만족할 때까지) — [요리 가이드](/guides/cooking-guide)에서 레시피 확인
- **환경을 선호에 맞게 조정** (따뜻함/시원함, [습도](/guides/how-to-raise-humidity), 밝기/어둡기)
- **집 짓기** (서식지 업그레이드)
- **매일 요청 수행** — 포켓몬에게 말을 걸어 「안락도 어때?」 선택 (하루 1회)

### 3. 포켓몬 요청 완료

대기 중인 요청이 있는 포켓몬은 **머리 위에 아이콘**이 표시됩니다. 요청을 완료하면 안락도가 직접 올라갑니다. 요청은 매일 리셋되므로 매일 확인하세요.

### 4. 포켓몬센터로 가는 길 정비

포켓몬은 포켓몬센터를 방문하여 교류합니다. 서식지에서 센터까지 원활한 길을 만들면 안락도가 향상됩니다. 도로의 구멍을 메우고, 높낮이 차이에는 계단을 설치하세요.

### 5. 환경 복원

일반적인 환경 복원도 도움이 됩니다:

- **시든 풀, 꽃, 나무에 물 주기**로 되살리기
- **어울리지 않는 지형 블록** 제거
- **식물 심기**로 경관 개선

### 6. 전설·환상의 포켓몬 활용

전설·환상의 포켓몬은 일반 포켓몬보다 **환경 레벨에 미치는 영향이 더 큽니다**. 효율을 극대화하려면 이들의 안락도를 우선적으로 올리세요.

## 레벨별 해금 요소

환경 레벨이 올라갈수록 새로운 콘텐츠가 해금됩니다:

| 레벨 | 해금 내용 |
|------|----------|
| 2 | 상점·통신 기능; 새 구매 가능 아이템 추가 |
| 3 | 포켓몬센터 개장 키트 상점 등장 |
| 5 | 특별 아이템 보상; 스토리 진행 마일스톤 |
| 5 이상 | 포켓라이프 PC에 고난도 챌린지 추가 |
| 10 | 모든 상점 아이템 해금; 포켓몬센터 디자인 키트; 레시피 보상 |

### 스토리 진행

게임 엔딩을 보려면 다음 4개 지역에서 **환경 레벨 5**를 달성해야 합니다:

- **바싹바싹 황야** (Withered Wasteland)
- **우중충 해변** (Bleak Beach)
- **울퉁불퉁 산지** (Rocky Ridges)
- **반짝반짝 부유섬** (Sparkling Skylands)

### 마을별 레벨 5 보상

| 마을 | 보상 | 용도 |
|------|------|------|
| 바싹바싹 황야 마을 | 폭죽 ×2 | 유니온 챌린지 EX |
| 울퉁불퉁 산지 마을 | 냉장고 | 유니온 챌린지 #7 |
| 우중충 해변 마을 | 세탁기 | 유니온 챌린지 #7 |
| 반짝반짝 부유섬 마을 | 게임보이 | 유니온 챌린지 #7 |

## 환경 레벨이 내려가는 원인

주의 — 다음 행동은 환경 레벨을 **낮출** 수 있습니다:

- **[포켓몬을 다른 마을로 이동](/guides/how-to-move-house)**하면 해당 포켓몬의 안락도 기여분 제거 (안락도가 높을수록 영향 큼)
- **부서진 가구**를 수리하지 않으면 안락도 하락
- **불 관련 오브젝트** (횃불, 캠프파이어 등)가 비에 꺼짐 — 지붕을 설치하여 보호하세요
- 지역에서 **포켓몬 제거** 시 총점이 떨어져 해금된 콘텐츠가 다시 잠길 수 있음

## 효율적인 레벨업 팁

1. **서식지를 포켓몬센터 근처에 모으기** — 가구 하나로 여러 포켓몬에게 효과를 줄 수 있습니다
2. **레벨 3에서 포켓몬센터 개장** — 개장 후 센터는 포켓몬 활동을 촉진합니다
3. **매일 안락도 확인** — 「안락도 어때?」 대화로 각 포켓몬이 원하는 것을 정확히 파악
4. **전설·환상의 포켓몬 우선** — 안락도 1포인트당 환경 레벨 기여도가 더 큽니다
5. **도로 정비를 소홀히 하지 않기** — 간단한 안락도 업으로 지역 내 모든 포켓몬에 효과
6. **불 오브젝트를 비로부터 보호** — 지붕이나 쉘터를 만들어 비 올 때 안락도 하락 방지
