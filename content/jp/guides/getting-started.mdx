# Pokopia 初心者ガイド

Pokopia の世界へようこそ！このガイドでは、ゲームのコアメカニクスを素早く理解し、冒険を良いスタートで始められるようお手伝いします。

## スターターパートナーを選ぼう

ゲーム開始時、3匹のポケモンの中から1匹をスターターパートナーとして選べます：

- **ほのおタイプ** — 攻撃が好きなトレーナー向き
- **みずタイプ** — 攻守バランスの取れた選択
- **くさタイプ** — サポートと持久戦が得意

> ヒント：どのスターターポケモンを選んでも、その後のゲーム体験には影響しません。一番好きなポケモンを選びましょう！

## バトルシステムの基本

Pokopia はターン制バトルシステムを採用しています。各ターンで以下のことができます：

1. **技を使う** — 各ポケモンは最大4つの技を覚えられます
2. **ポケモンを交代する** — タイプ相性に応じて柔軟に切り替え
3. **アイテムを使う** — HP回復、状態異常の治療など
4. **逃げる** — 野生ポケモンバトルのみ

### タイプ相性

タイプ相性はバトルの核心です。例えば：

| 攻撃タイプ | 効果ばつぐん | 効果いまひとつ |
|-----------|------------|-------------|
| ほのお | くさ、こおり、むし、はがね | みず、ほのお、いわ、ドラゴン |
| みず | ほのお、じめん、いわ | みず、くさ、ドラゴン |
| くさ | みず、じめん、いわ | ほのお、くさ、どく、ひこう、むし、ドラゴン、はがね |

## Pokopia の世界を探索しよう

Pokopia には豊かなエリアが探索を待っています：

- **エメラルド平原** — 初期エリア、レベル上げに最適
- **アズールコースト** — みずタイプポケモンの楽園
- **ブレイズ火山** — 高レベルチャレンジエリア
- **スターライトフォレスト** — レアポケモンが潜む

## 次のステップ

基本を覚えたら、以下をおすすめします：

1. エメラルド平原で異なるタイプのポケモンを数匹捕まえる
2. タイプバランスの取れたチームを編成する
3. 最初のジムに挑戦する
4. 上級ガイドでさらなるテクニックをチェックする
